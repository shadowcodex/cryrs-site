Here are the instructions :)

Tables

  • The tables are dynamically generated, and all data is lost upon refresh.
  • The system index table can be sorted by clicking on the table headers.
  • All data is as realtime as the sources are. AKA Kills / zkillboard, FW Stats = every hour
  • There is no need to refresh the page as data will refresh automatically.

Killmails

  • They are as realtime as they show up on zkillboard. Make sure to pay attention to the time.
  • Clicking on ship icon will take you to the killmail.
  • Clicking on corp icon next to pilot will take you to that corp's killboard on zkill.
  • Upon a kill happening, the system it happened in will flash.
  • All kills shown are only kills that happen within a system on the map.
  • Hovering over a row will highlight the system on the map.
  • Number shown in the parenthesis is the number of attackers.
  • Kill Colors:
    • RED: means that a galmil pilot was killed.
    • GREEN: means that a galmil pilot killed someone.
    • YELLOW: means that a calmil pilot killed someone that wasn't galmil.
    • BLUE: means that a calmil pilot was killed by someone that wasn't galmil.

Map Network

  • Each node represents a system.
  • Shows system name
  • Shows gate connections
  • Shows ownership
  • Shows contested percentage
  • Show whether the last 3 hours of change was positive, negative, or stable
  • You can zoom and pan the map
  • Shows regions and what systems are in each region
  • Shows the division of the North and South warzones using a red line
  • Is only of the Gallente and Caldari warzone
  • System Colors:
    • GREEN: means that a the system is occupyed by the Gallente.
    • BLUE: means that the system is occupyed by the Caldari.
    • YELLOW: means that the system a high security system.
    • PURPLE: means that the system a low sec or null sec system that isn't part of Faction Warfare.
    • RED OUTLINE: means that the system in question is a known gate camp system.

Missions

  • Limited mission information is shown on the map network.
  • GREEN: Gallente Missions = Federal Defense Union
  • BLUE: Caldari Missions = State Protectorate

System Indexes

  • Shows system name
  • Shows contested percentage
  • Shows rate of change within last 3 hours
  • Shows who is occupying the system at that momement.
  • Hovering over a row will highlight the system on the map
  • Double clicking a row will open that systems stats on dotlan.
  • Index Colors:
    • GREEN: means that a the system is occupyed by the Gallente.
    • BLUE: means that the system is occupyed by the Caldari.

Kill / Death Stats

  • Kill Stats shows the number of kills each faction has gotten. AKA Gallente kills a Caldari, then the Gallente kill stats goes up by one.
  • Death Stats shows the number of deaths each faction has gotten. AKA Gallente pilot dies, then the Gallente death stats goes up by one.
